Bienvenido a StatView Pro

StatView Pro es una aplicación de análisis estadístico completa inspirada en SPSS, que funciona completamente en tu navegador usando SQLite como motor de base de datos.

Comandos de Sintaxis

DESCRIPTIVES var1 var2 var3
FREQUENCIES var1
TTEST GROUPS=grupo(1 2) /VARIABLES=score
ANOVA score BY grupo
CORRELATIONS var1 var2 var3
REGRESSION /DEPENDENT=y /METHOD=ENTER x1 x2
CROSSTABS var1 BY var2
CHISQUARE var1 var2

Características Implementadas

Módulo Descripción
Base de datosSQLite vía SQL.js (WebAssembly) — completamente en cliente, sin servidor
ImportaciónExcel (.xlsx), CSV, SQLite (.db / .sqlite / .sqlite3)
ExportaciónExcel, CSV, SQLite (.db), HTML (output), PNG (gráficos)
Vista de DatosTabla editable tipo spreadsheet con navegación por teclado (flechas, Tab)
Vista de VariablesDefinición de tipos (numérico, entero, cadena, fecha), etiquetas, valores, medida (nominal / ordinal / escala)
DescriptivosMedia, mediana, moda, desviación estándar, varianza, mínimo, máximo, rango, percentiles (Q1, Q3, IQR, P5, P10, P90, P95), asimetría (skewness), curtosis (kurtosis)
T-TestOne-sample, independent samples (muestras independientes), paired samples (muestras pareadas) con intervalos de confianza
ANOVAUnidireccional (one-way) con tabla completa, eta cuadrado (tamaño del efecto)
CorrelacionesPearson (paramétrico) y Spearman (no paramétrico) con matriz completa, valores p e interpretación
RegresiónLineal múltiple (OLS) con coeficientes, errores estándar, t, valores p, R², R² ajustado, ANOVA del modelo y ecuación
Chi-CuadradoPrueba de independencia con tablas de contingencia, frecuencias observadas y esperadas
FrecuenciasTablas de frecuencia con porcentajes, acumulados, gráficos de barras y pastel opcionales
Tablas CruzadasContingencia con chi-cuadrado integrado, porcentajes y frecuencias esperadas
GráficosHistograma, boxplot, dispersión (scatter), barras, pastel (pie), líneas — generados con Chart.js
SintaxisEditor de comandos tipo SPSS ejecutable línea por línea: DESCRIPTIVES, TTEST, ANOVA, CORRELATIONS, REGRESSION, FREQUENCIES, CROSSTABS, CHISQUARE
TransformacionesLogaritmo natural, logaritmo base 10, raíz cuadrada, cuadrado, recíproco, Z-score (estandarizar), valor absoluto, fórmula personalizada
Datasets múltiplesGestión de múltiples datasets simultáneos con conmutación rápida
ResponsiveDiseño adaptativo completo para móvil, tablet y escritorio con menú hamburguesa
Atajos de tecladoCtrl+O (importar), Ctrl+S (exportar Excel), Ctrl+R (ejecutar sintaxis), Ctrl+1–5 (cambiar pestañas)

Tecnologías utilizadas

HTML5 • CSS3 • JavaScript (Vanilla) • SQL.js (SQLite WebAssembly) • SheetJS (xlsx.js) • Chart.js • Font Awesome
